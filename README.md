This repository is designed for beginners who are learning Python fundamentals through simple, easy-to-understand programs. It focuses on building strong programming logic and problem-solving skills using core Python concepts.

The goal is to help learners understand how to think in Python, not just write code.

Topics Covered

This repository includes hands-on examples of the following Python fundamentals:

📋 Lists
📖 Dictionaries
🔢 Sets
📦 Tuples
✂️ Slicing
🔁 Recursion
